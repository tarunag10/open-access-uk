# Repository Map

## Public Tools

- `open-access-uk-site`: umbrella site and public front door.
- `letter-generator`: reasonable-adjustment and public-service letter drafting.
- `accessible-forms`: public-service form patterns and remediation reports.
- `public-service-directory`: complaint and escalation route planning.
- `legal-templates`: plain-English template drafting and bundles.
- `design-system`: accessible tokens, patterns, and component recipes.

## Parent Tooling

- `contributor-tools/maintainer-helper`: local helper for repository readiness,
  contributor onboarding, labels, issue templates, and governance signals.

## Root Files

- `.github/`: governance templates and workflows.
- `data/`: source and repository registries.
- `docs/`: architecture, accessibility, privacy, release, and provenance docs.
- `scripts/`: suite verification and validation gates.
- `shared/`: shared static assets such as design tokens.
